# markdown-py

Python tool to convert Markdown files to HTML files.

*Very work-in-progress! Expect breaking changes.*

## Example usage

**markdown file**
```
# Hello, World!
This is some text.
```

**html file**
```html
<h1>Hello, World!</h1>
<p>This is some text.</p>
```

**python (running on Linux)**
```
python3 markdown.py input.md output.html
```
